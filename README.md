# meegle-cli

A TUI for managing Meegle tasks in the terminal.

## Overview
This project interacts with the Meegle API to allow users to manage their tasks directly from the command line using a rich Terminal User Interface.

## Getting Started

### Prerequisites
- Go 1.20+

### Installation
```bash
git clone https://github.com/ningzio/meegle-cli.git
cd meegle-cli
go run cmd/meegle/main.go
```

## Development
- Uses `golangci-lint` for linting.
- Follows the Bubble Tea MVU architecture.
